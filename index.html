1. Weibull 分佈基本原理

威布爾分佈 (Weibull Distribution) 是可靠度工程中最通用的機率模型。其核心優勢在於能通過形狀參數 (Beta) 的變化，模擬出產品生命週期中不同的失效特徵。

核心數學模型

可靠度函數 R(t) 表示在時間 t 仍然存活的機率：

\( R(t) = e^{-(t/\eta)^\beta} \)

其中 $\beta$ 為形狀參數，$\eta$ 為尺度參數（特徵壽命）。

基礎數據類型

  • 失效數據 (Complete/Failure Data): 觀測到產品確切發失效的時間點。
  • 右設限數據 (Right Censored/Suspended): 測試結束時產品仍完好，僅知其壽命大於當前時間。

2. 分佈參數深度解析

形狀參數 Shape (Beta β)

Beta 決定了失效分佈的幾何趨勢，直接反應「失效機制」：

Beta 數值 失效機制 (Mechanism) 工程特徵
β < 1.0 早期失效 (Infant Mortality) 失效率隨時間下降。常與設計缺陷、製造髒污或材料瑕疵有關。
β = 1.0 隨機失效 (Random Chance) 失效率為常數。失效由隨機且不可預期的外部應力引起。
1.0 < β < 4 磨耗失效 (Wear-out) 失效率隨時間增加。代表產品進入健康衰減期，如磨損、疲勞。
β > 4.0 快速劣化 (Rapid Aging) 失效極度集中。代表材料或結構在短時間內徹底崩溃。

尺度參數 Scale (Eta η)

又稱「特徵壽命」。代表累積失效機率達到 63.2% 的時間點。它是衡量產品壽命長度的主指標。

3. B-Life 工程指標意義

B-Life 指標代表「失效百分比達到 x% 時的時間點」。

指標 失效比例 工程應用場景
B1 1% 失效 航空航天、醫療植入裝置、汽車安全氣囊等極高安全性組件。
B10 10% 失效 工業標準。常用於軸承、電機與一般消費品保固期限設定。
B50 50% 失效 中位數壽命。用於衡量整批產品的平均能量水平。

計算模型

\( t = \eta \cdot [-\ln(R)]^{1/\beta} \)

4. 樣本數量與信心度建議

樣本量直接決定了分析結果的精確度與統計信心水平：

極低預測力 3-5 顆

僅能觀察初步區間，Beta 值波動极大。

初步分析 6-10 顆

行業內初步評估 B10 之低點數建議值。

專業工程建議 12-20 顆

取得穩定參數與窄信心區間的黃金區間。

Python 自動化分析 (範例碼)

import numpy as np
from scipy import stats

# 工程觀測壽命數據 (失效點)
lifetimes = [450, 580, 690, 780, 870, 950]

# Weibull 參數擬合
# floc=0 代表位置參數為0 (兩參數 Weibull)
shape, loc, scale = stats.weibull_min.fit(lifetimes, floc=0)

print(f"Shape (Beta): {shape:.4f}")
print(f"Scale (Eta): {scale:.2f}")
